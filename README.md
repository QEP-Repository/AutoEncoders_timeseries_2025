# AutoEncoders_timeseries_2025

This github will host the codes for the paper "On the Use of Autoencoders to Study the Dynamics and the Causality Relations of Complex Systems with Applications to Nuclear Fusion" by R.Rossi, A.Murari, T.Craciunescu, N.Rutigliano, I.Wyss, J.Vega, P.Gaudio and M.Gelfusa, on behalf of JET contributors and EUROfusion Tokamak Exploitation Team once the paper is accepted for publication.
